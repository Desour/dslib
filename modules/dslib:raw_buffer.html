Class RawBuffer

A byte-addressable secure wrapper for a cdata buffer.

Maximum size is currently about 0x1p60 bytes.

Integer types can be numbers or 64 bit LuaJIT cdata integers (ie. 1ULL).

Note: The API is very unstable.
